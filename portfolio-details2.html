Information sur le projet

  • Categorie: Developpement web
  • Client: OOREDOO Tunisie
  • Date de projet : 30 Avril, 2019
  • Projet: Systéme gestionnaire de balise

Ce système permet à travers d'une interface de gérer les balises, c’est à dire ajouter, modifier ou supprimer une balise. Chaque balise ajoutée permet d’activer automatiquement la transaction client à suivre, contrairement aux solutions existantes qui nécessite l’ajout manuel d’un code propriétaire dans les pages web. Notre système permet aussi la gestion des utilisateurs en attribuant les droits d’ac- cès approprié à chaque utilisateur. Grâce à ce système de gestion de balises les deux fonctionnalités suivantes sont assurées. — La traçabilité des transactions des clients. — Le calcul des statistiques qui décrivent le comportement de la clientèle.
